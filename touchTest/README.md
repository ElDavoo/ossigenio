## touchTest
Just a simple sketch for ESP32-WROOM-32D to retrieve raw values from touch pin.
See image for pin reference.
|Name   |Pin name   |Pin number|Feedback|
|---|---|----|---|
|T0|GPIO4|4|Positive|
|T1|GPIO0|NOT USED|
|T2|GPIO2|2|Neutral|
|T3|MTDO|15|
|T4|MTCK|13|Negative|
|T5|MTDI|12|
|T6|MTMS|17|
|T7|GPIO27|27|
|T8|32K_XN|33|
